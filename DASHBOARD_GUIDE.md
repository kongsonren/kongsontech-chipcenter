## 🚀 快速开始

### 步骤 1：安装 Streamlit

**双击运行**：
```
install-streamlit.bat
```

等待安装完成（约 2-5 分钟，需要联网）

### 步骤 2：启动看板

**双击运行**：
```
start-dashboard.bat
```

浏览器会自动打开：http://localhost:8501

### 步骤 3：开始使用

1. 上传 PDF 规格书
2. 点击"开始解析"
3. 添加到产品库
4. 一键上架到 1688

## 📊 功能介绍

### 1️⃣ 总览看板
- 今日作战态势
- KPI 核心指标
- 产品状态分布
- 销售趋势图

### 2️⃣ PDF 规格书解析
- 支持芯片规格书、报价单、产品参数表
- 自动提取关键字段（型号、电压、功率、成本等）
- 一键添加到产品库

### 3️⃣ 1688 上架监控
- 产品列表管理
- 上架状态追踪
- 一键上架操作
- 审核状态监控

### 4️⃣ 销售数据
- 实时销售额
- 订单统计
- 转化率分析
- 多平台数据（1688/淘宝/京东/拼多多）

### 5️⃣ 团队任务
- 成员状态管理（中山/华强北/宁波）
- 任务分配与追踪
- 待办/已完成看板

### 6️⃣ 自动化流程
- 全生命周期追踪
- PDF→解析→上架→销售→同步
- 定时任务配置

## ❓ 常见问题

### Q1: 双击 bat 文件没反应？
**A:** 请检查：
1. Python 是否安装在 `C:\Users\59605\AppData\Local\Python\pythoncore-3.14-64\`
2. 是否已安装 Streamlit（运行 `install-streamlit.bat`）

### Q2: 浏览器无法打开？
**A:** 手动访问 http://localhost:8501

### Q3: PDF 解析失败？
**A:** 当前为演示版本，完整解析功能需要：
- 提供 PDF 样品
- 定制解析规则
- 配置 OCR 引擎

### Q4: 如何连接 1688 API？
**A:** 需要：
1. 1688 开放平台账号
2. 申请 API Key
3. 配置 OAuth 认证
4. 我可以帮您对接（告诉我）
