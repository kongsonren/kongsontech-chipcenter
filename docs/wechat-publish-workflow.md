## 🎯 功能概述

从 PDF 规格书到微信公众号发布的**全自动工作流**：

```
PDF 规格书 → 数据提取 → AI 文案生成 → 自动发布 → 人工审核 → 发布
```

## 🚀 快速开始

### 第一步：启动看板

双击运行：
```
start-dashboard.bat
```

浏览器打开：`http://localhost:8501`

### 第二步：上传 PDF 规格书

1. 选择左侧菜单 **"📄 PDF 规格书解析"**
2. 上传产品 PDF 规格书（如 `LM2596.pdf`）
3. 等待自动解析（约 10-30 秒）
4. 查看解析结果并确认

### 第三步：生成文案

1. 选择左侧菜单 **"📱 公众号发布"**
2. 从下拉列表选择刚上传的产品
3. 选择内容类型：
   - **公众号推文**（推荐）
   - 知乎技术文
   - 朋友圈文案
   - 视频脚本
   - 1688 详情页
4. 点击 **"🪄 一键生成文案"**

### 第四步：预览和导出

AI 会自动生成文案，您可以：
- **预览**: 直接在页面查看生成的内容
- **导出**: 点击"导出为 Markdown"保存为文件
- **复制**: 点击"复制到剪贴板"手动粘贴

### 第五步：发布到公众号

1. 点击 **"🚀 发布到公众号"** 展开选项
2. 点击 **"🌐 启动发布助手"**
3. 浏览器自动打开公众号后台
4. **扫码登录** 您的企业公众号
5. 系统自动填充：
   - 文章标题
   - 正文内容
   - 摘要（可选）
6. **人工审核**: 仔细检查内容准确性
7. 调整排版和图片位置（如需要）
8. 点击 **"发表"** 按钮

## 🔧 技术说明

### RPA 自动化原理

使用 **Playwright** 浏览器自动化框架：
- 模拟真实用户操作
- 自动登录公众号后台
- 自动填充表单内容
- 自动上传图片

### 为什么采用 RPA + 人工审核？

| 方案 | 优点 | 缺点 |
|------|------|------|
| **RPA+ 人工审核** | 兼容性好，无需 API 权限，可把控内容质量 | 需要人工参与 |
| 企业微信 API | 全自动，无需人工 | 需要申请权限，审核周期长 |
| 纯手动 | 完全可控 | 效率低，重复劳动 |

**结论**: RPA+人工审核最适合您当前的团队配置！

## ⚠️ 注意事项

### 内容审核
- ✅ **必须人工审核**后才能发布
- ✅ 检查技术参数是否准确
- ✅ 检查联系方式是否正确

### 发布频率建议
- **新品发布**: 每周 1-2 篇
- **技术分享**: 每周 1 篇
- **客户案例**: 每 2 周 1 篇

### 封号风险规避
- 不要使用外挂/破解工具
- 发布间隔不要太短（建议>5 分钟/篇）
- 不要批量注册/操作多个公众号
- 遵守微信平台规则

## 🆘 常见问题

### Q1: 发布助手启动失败？
**A**: 检查以下几点：
1. Python 是否已安装（3.8+）
2. 是否安装了 playwright: `pip install playwright`
3. 是否安装了浏览器：`playwright install chromium`

### Q2: 无法自动填充内容？
**A**: 公众号后台可能更新了界面，请联系软件工程师检查选择器。

### Q3: 扫码登录超时？
**A**: 网络问题，请重新启动发布助手再试。

### Q4: 生成的文案不满意？
**A**: 可以：
1. 手动编辑修改
2. 重新选择内容类型
3. 提供更多产品数据给 AI
