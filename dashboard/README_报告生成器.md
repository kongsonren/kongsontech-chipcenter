# 冠辰科技作战指挥中心 - 静态 HTML 报告版本

## 📋 文件说明

| 文件 | 用途 |
|------|------|
| `chip_report_generator.py` | 静态 HTML 报告生成器（新版本） |
| `chip_dashboard.py` | Streamlit 互动看板（原版） |

---

## 🚀 使用方法

### 方式一：直接运行生成报告

```bash
cd C:\projects\chip-automation-workflow\dashboard
python chip_report_generator.py
```

**运行后会：**
1. 自动从数据库加载产品数据
2. 在 `reports/` 目录生成 HTML 报告
3. 文件名格式：`chip_dashboard_20260323_153045.html`

### 方式二：定时任务自动运行

如果您想每天自动生成报告，可以设置定时任务：

```bash
# 每天上午 9 点自动生成报告
0 9 * * * cd C:\projects\chip-automation-workflow\dashboard && python chip_report_generator.py
```

---

## 📁 输出位置

生成的报告保存在：
```
C:\projects\chip-automation-workflow\dashboard\reports\
```

直接用浏览器打开即可查看，**无需启动任何服务器**。

---

## ✨ 功能特点

### ✅ 优势
- **无需启动服务器** - 不像 Streamlit 需要一直运行
- **文件可分享** - HTML 文件可以发送给团队成员
- **可打印** - 支持打印成 PDF 报告
- **响应式设计** - 手机/电脑都能看
- **离线查看** - 生成后不需要网络

### ⚠️ 限制
- **数据不实时** - 需要重新运行脚本才能更新数据
- **无交互功能** - 不能编辑产品或上传 PDF（只读）

---

## 📊 报告内容

1. **KPI 指标卡** - 产品总数、已上架、待上架、审核中
2. **产品数据表** - 所有产品的详细信息
3. **状态统计** - 各状态产品数量

---

## 💡 建议使用场景

**使用 HTML 报告版本：**
- 每天/每周固定时间查看数据
- 需要发送给团队成员
- 需要打印或保存为 PDF
- 不想一直运行服务器

**使用 Streamlit 版本：**
- 需要频繁编辑产品数据
- 需要上传和解析 PDF
- 需要实时查看最新数据
- 需要交互式操作
