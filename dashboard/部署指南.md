## 📦 部署方案

### 方案一：Docker 部署（推荐）

#### 1️⃣ 云服务器准备

**推荐云服务商：**
- 阿里云（性价比高）
- 腾讯云（国内速度快）
- 华为云（企业级）
- AWS（国际化）

**最低配置：**
- CPU: 2 核
- 内存：4GB
- 硬盘：40GB SSD
- 系统：Ubuntu 22.04 LTS

#### 2️⃣ 上传代码到服务器

```bash
# 在本地电脑执行（使用 Git）
cd C:\projects\chip-automation-workflow
git init
git add .
git commit -m "Initial commit"

# 然后推送到 GitHub 或 Gitee
# 或者直接用 SCP 上传
scp -r dashboard/* root@你的服务器 IP:/root/dashboard/
```

#### 3️⃣ 在服务器上执行

```bash
# 登录服务器后
cd /root/dashboard

# 构建 Docker 镜像
docker-compose build

# 启动服务
docker-compose up -d

# 查看运行状态
docker-compose ps
```

#### 4️⃣ 配置防火墙

在云服务商控制台开放端口：**8501**

### 方案二：直接部署（无 Docker）

```bash
# 1. 安装 Python
sudo apt update
sudo apt install python3 python3-pip -y

# 2. 安装依赖
pip3 install streamlit pandas reportlab PyPDF2 psutil schedule

# 3. 上传代码
# 使用 SCP 或 Git 上传 dashboard 目录

# 4. 启动服务
cd /root/dashboard
streamlit run chip_dashboard.py --server.port 8501 --server.address 0.0.0.0

# 5. 后台运行（使用 systemd）
sudo nano /etc/systemd/system/streamlit.service
```

**systemd 服务配置：**
```ini
[Unit]
Description=Streamlit Dashboard
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/dashboard
ExecStart=/usr/bin/python3 -m streamlit run chip_dashboard.py --server.port 8501 --server.address 0.0.0.0
Restart=always

[Install]
WantedBy=multi-user.target
```

**启动服务：**
```bash
sudo systemctl daemon-reload
sudo systemctl enable streamlit
sudo systemctl start streamlit
sudo systemctl status streamlit
```

## 🔐 安全配置

### 1. 添加密码保护

在 `chip_dashboard.py` 中添加：

```python
import streamlit as st

# 简单的密码保护
PASSWORD = "Kongson2026"  # 修改为你的密码

def check_password():
    def password_entered():
        if st.session_state["password"] == PASSWORD:
            st.session_state["password_correct"] = True
            del st.session_state["password"]
        else:
            st.session_state["password_correct"] = False

    if "password_correct" not in st.session_state:
        st.text_input(
            "密码：",
            type="password",
            on_change=password_entered,
            key="password"
        )
        return False
    elif not st.session_state["password_correct"]:
        st.text_input(
            "密码错误，请重试：",
            type="password",
            on_change=password_entered,
            key="password"
        )
        return False
    else:
        return True

if check_password():
    # 原有代码放在这里
    main()
```

### 2. 使用 HTTPS（推荐）

用 Nginx 反向代理：

```nginx
server {
    listen 443 ssl;
    server_name dashboard.kongson.com;

    ssl_certificate /etc/ssl/certs/your_domain.crt;
    ssl_certificate_key /etc/ssl/private/your_domain.key;

    location / {
        proxy_pass http://localhost:8501;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
